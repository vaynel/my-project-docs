# ReactFlow
## 개요 

ReactFlow(이전 명칭: react-flow-renderer)는 대화형 노드 기반 편집기, 다이어그램, 시각적 애플리케이션을 구축하기 위한 고도로 사용자 정의 가능한 React 라이브러리입니다.

주요 특징:
- 노드와 엣지(연결선)를 기반으로 한 그래프/플로우차트 시각화
- 드래그 앤 드롭, 확대/축소, 패닝 등 기본 인터랙션 제공
- 커스텀 노드와 엣지 타입 지원
- TypeScript 완전 지원
- 성능 최적화 (가상화를 통한 대규모 그래프 처리)

**현재 패키지명**: `@xyflow/react` (v12+, 2025년 3월 리브랜딩)
**이전 패키지명**: 
- `reactflow` (v11, 2022년 9월 ~ 2025년 3월)
- `react-flow-renderer` (v10 이하)

## 역사 

- **2019년**: react-flow-renderer로 시작
- **2021년**: v10 릴리스, 안정화 및 성능 개선
- **2022년 9월**: v11 릴리스, 패키지명을 `reactflow`로 변경
- **2025년 3월**: 패키지명을 `@xyflow/react`로 리브랜딩 (xyflow 브랜드 통합)
- **현재**: 지속적인 업데이트 및 커뮤니티 지원

**참고**: 현재 프로젝트에서는 `reactflow: 11.11.4`를 사용 중이며, 이는 `@xyflow/react`로 리브랜딩되기 전의 버전입니다.

## 장/단점
### 장점 

1. **사용 편의성**
   - React 컴포넌트 기반으로 직관적인 API 제공
   - 풍부한 기본 기능 (드래그, 줌, 패닝 등)
   - TypeScript 완전 지원

2. **커스터마이징**
   - 커스텀 노드 타입 정의 가능
   - 커스텀 엣지 타입 정의 가능
   - 스타일링 완전 제어 가능

3. **성능**
   - 가상화를 통한 대규모 그래프 처리
   - 보이는 영역만 렌더링하여 최적화
   - 부드러운 애니메이션 지원

4. **생태계**
   - 활발한 커뮤니티
   - 상세한 공식 문서
   - 다양한 예제 제공

5. **유연성**
   - 다양한 레이아웃 알고리즘 지원
   - 이벤트 핸들링 세밀한 제어
   - 플러그인 시스템

### 단점

1. **학습 곡선**
   - 초기 설정이 다소 복잡할 수 있음
   - 노드/엣지 커스터마이징 시 개념 이해 필요

2. **번들 크기**
   - 기능이 많아 상대적으로 큰 번들 크기
   - 트리 쉐이킹으로 일부 완화 가능

3. **복잡한 레이아웃**
   - 자동 레이아웃 기능이 제한적
   - 복잡한 그래프의 경우 추가 라이브러리 필요할 수 있음

4. **모바일 지원**
   - 터치 제스처 지원은 있으나 데스크톱에 최적화
   - 모바일에서의 사용성은 추가 작업 필요

## 비슷한 라이브러리

| 라이브러리 | 언어/프레임워크 | 특징 | 장점 | 단점 |
|-----------|---------------|------|------|------|
| **ReactFlow** | React | 노드 기반 그래프 편집기 | 커스터마이징 우수, TypeScript 지원 | 학습 곡선 존재 |
| **D3.js** | Vanilla JS | 데이터 시각화 라이브러리 | 매우 유연하고 강력함 | 낮은 수준의 API, 학습 곡선 높음 |
| **Cytoscape.js** | Vanilla JS | 그래프 이론 라이브러리 | 레이아웃 알고리즘 풍부 | React 통합 시 추가 작업 필요 |
| **vis.js** | Vanilla JS | 네트워크 시각화 | 빠른 렌더링 | React 통합 시 추가 작업 필요 |
| **Mermaid** | Markdown | 다이어그램 생성 | 텍스트 기반, 간단함 | 인터랙티브 제한적 |
| **JointJS** | Vanilla JS | 다이어그램 라이브러리 | 강력한 기능 | React 통합 복잡, 상업용 라이선스 |
| **GoJS** | Vanilla JS | 상업용 다이어그램 라이브러리 | 매우 강력한 기능 | 상업용 라이선스 필요 |
| **React D3 Tree** | React | 트리 구조 시각화 | React 네이티브 | 트리 구조에 특화됨 |
| **React Graph Vis** | React | vis.js의 React 래퍼 | vis.js 기능 활용 | 커스터마이징 제한적 |

**선택 기준:**
- React 프로젝트 + 노드 기반 편집기 필요 → **ReactFlow** 추천
- 복잡한 레이아웃 알고리즘 필요 → **Cytoscape.js** 고려
- 단순 다이어그램 생성 → **Mermaid** 고려
- 상업용 프로젝트 + 고급 기능 필요 → **GoJS** 고려
