# 환경설정
## node 20.x 사용
- Next.js 16.0.7은 Node 20 버전 이상에서 사용가능함 
### Node 20 Install
```bash
# NVM 설치 (아직 없다면)
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 터미널 재시작 또는
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"

# Node.js 20 설치 및 사용
nvm install 20
nvm use 20
nvm alias default 20

# 확인
node --version

# 프로젝트 의존성 재설치
cd frontend
rm -rf node_modules package-lock.json
npm install
```
